# raw/ — Immutable source material

Drop original sources here: papers/articles (converted to markdown where
possible), exported threads, tickets, meeting notes, datasets, and screenshots
(in `assets/`).

**Rules**
- Never edit files in this folder — they are provenance.
- Summarize sources in `../summaries/`, then synthesize into
  `../concepts/` and `../entities/` pages.
- Reference a raw file from a page via the `sources:` frontmatter and a relative
  link, e.g. `[raw/thread.md](../raw/thread.md)`.
